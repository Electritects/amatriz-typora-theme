# AMatriz Artwork

Artwork prepared for Typora Theme Gallery submission.

## Gallery Thumbnail

- `amatriz.png` - 250x200 PNG, ready to copy into Typora Theme Gallery's `thumbnails` directory.
- `amatriz-500x400.png` - 500x400 PNG, higher-resolution thumbnail variant.
- `amatriz-preview.png` - 1200x960 preview image for README, release notes, or pull request discussion.
- `amatriz-thumbnail-source.svg` - editable source artwork for the thumbnail.

Typora's submit guide asks for a 250x200 or 500x400 thumbnail in the gallery `thumbnails` directory, with the filename listed in the post markdown.
